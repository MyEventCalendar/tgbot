# tgbot


### Create virtual environment
```shell
python3 -m venv env
```

### Activate virtual environment
```shell
. ./env/bin/activate
```

### Install packages from requirements.txt
```shell
pip install -r requirements.txt
```

### Export environment variables
```shell
export API_KEY=YOUR_API_KEY API_URL=YOUR_API_URL
```

### Start main.py
```shell
python main.py 
```
